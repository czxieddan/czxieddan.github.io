# Security Policy

## Supported Versions

The **latest version** that has been released is **v0.1.0-alpha**, below are the currently supported versions, versions that are **too old** are **not supported**.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x-alpha   | :white_check_mark: |
| < 0.1.x-alpha   | :x:                |

TEASER - 西欧局势

1. 日志简述

    这是我们初次公开发布本作(A Pathetic Epoch:Rejuvenation In Purgatory)的制作内容，我们对此深感激动不已。但同时可能有诸多疏漏与缺陷有待改进，还望多多谅解。如果您有任何想法，欢迎您联系我们给与建议或加入我们的制作。

    本次内容篇幅可能稍长，因为我们需要在对内容的介绍中夹杂部分对我们作品世界观设定的介绍。为不让您变得无聊，我们为该页面播放了一首本作内某一地区的主题曲(我们会不定期为当前界面替换音频)，若您并不喜欢它可以将其关闭。

    我们深知许多人并没有曾了解过我们的世界观，许多背景故事设定可能令人感到陌生。如果您对此很感兴趣，可以点击下方的按钮跳转到我们的世界观介绍页面。

2. 日志正文

    让我们进入正题，本次内容我们为大家带来了西欧地区(广义)的局势介绍，让各位能够从中充分感受到我们作品中“Rejuvenation In Purgatory”的矛盾构成。虽然这仅仅包含了对各势力的文本介绍，并未能为大家带来丰富的美术内容，但相信在不远的将来它们就将出现！

    1. 局势概述

      进入21世纪后，西欧已不再是那个以秩序、理性与繁荣自我定义的大陆。

      “无人胜利的战争”结束后，和平并未降临，国家这一概念反而开始解体，条约与议会沦为档案，权力碎裂为彼此冲突的解释体系，而合法性则被交付给神话与极端信仰。

      德意志地区成为这一进程最残酷的实验场：柏林沉没，联邦消失，地方势力在真空中争夺“何为德国”的解释权，王权复辟者、工人自治、宗教国家与毁灭主义者并立其间，使这里不再构成国家，而成为向全欧洲开放的政治炼狱……

      法兰西选择强行延续秩序，在高度集权下维持国家连续性，并以军事管制将低地纳入缓冲区，它不再输出自由与革命，而是输出一种仍在运转、却已脱离初衷的秩序……

      亚平宁半岛分裂，以各异身形响应着闭幕：钢铁领主、宗教狂热、死亡美学与末日狂欢并存，使未来不再是目标，而成为被消费的体验……

      伊比利亚沉溺于历史回声之中，王权崩解，复辟、圣战与航海幻影彼此对峙，每一次复兴都指向对现实的逃避……

      不列颠尼亚在孤立中分裂，军政委员会取代王权，并试图以科技与神话制造新的统治象征……

      西欧的现状并非异常，而是结果：

      当理性耗尽解释牺牲的能力，极端、神话与毁灭便成为可行的统治形式。

      这里没有真正的胜利者，只有在废墟中将生存称为重生、将炼狱误认为天国的人类；历史仍在继续——但它已不再为任何人负责。

    2. 分区介绍

      本次介绍的地区中势力较多，为了能清晰的展现出每个势力的特点以及其在地缘上的关系，我们采用了分区介绍的形式，分别向您介绍每个势力的开局情况以及特色内容。对于部分我们深有心得的势力，可能会加以编者的话形式的拓展说明，这可能包含其未来的更新方向或不易被觉察的细节。但相应的，受于制作精力有限与时间紧迫，部分势力可能暂无其特色的内容或不过多赘述，当然在后续推出相应内容后我们会加以补充，也就是说这个界面可能随时迎来它的更新。

      (对于下方展示的任意地区预览图片，出于性能考虑使用了压缩格式，您可以点击打开以预览其原图或下载至本地查看。)

      1. 亚平宁半岛

        南欧的热土，曾是帝国伊始之地，也曾孕育伟大的文艺复兴思潮。这里一度是世界的中心，中国的丝绸，印度的香料，埃及的粮食都在这里汇聚。没有什么东西在罗马买不到，如果有，那一定不存在这个世界上。可那早已是历史的劫灰。强盛的帝国毁灭于自大和奢靡。历经千年浮沉，二十一世已经过半……罗马政府已经无力控制治下的各地区，各地已经处于半独立状态：维特鲁威的工业区，工人们崇拜着神圣的工业；陷入谵妄的梦呓之人，在末日前进行最后的狂欢；天主的国仍然行走于大地之上，可世上已无基督。文艺复兴的火焰再度燃烧，究竟是涅槃重生，或是化为灰烬，一切仍未可知……

        1. 意大利共和国

          名义上的中央政府仍高悬罗马的旗帜，却对广袤国土的掌控日渐松弛，权力如沙漏般流逝，各地自治的火焰悄然点燃，共和国已成为一个空壳般的象征，在风雨中摇曳。

        2. 意大利亚利古里亚热那亚自治共和国

          西北沿海的港口之珠摆脱中央羁绊自立为自治共和国，海风携带着自由的咸涩，热那亚的灯塔重新照亮属于自己的航道。

        3. 意大利亚两西西里自治共和国

          南部炽热的土地上，两西西里的古老血脉复苏，自治的旗帜在火山灰中猎猎作响，昔日王国的余辉在此重现，顽强守护着那份不屈的南方灵魂。

        4. 维特鲁威钢铁领

          盘踞于传统工业区的钢铁堡垒，这里保留了战火中幸存的熔炉与机器，文艺复兴的幽灵悄然复苏——血肉苦弱，机械飞升。简陋的义体工坊中，金属与血肉交织，脱离尘世的思潮如烟雾般弥漫，追逐着永恒力量的幻梦。

        5. 圣体真修会

          人们将所在山岭奉为圣山，虔诚宣扬“神迹”的降临，信徒们通过易北河大爆炸孕育的变异真菌致幻，声称能面见上帝。那狂热的信仰如烈火燃烧，吞噬理性，点亮黑暗中的幻影。

        6. 卡拉拉艺术静默体

          大理石矿脉的静默守护者，他们的意识形态如冰冷的石雕般坚硬——时间静止，思想永恒。死亡在此被颂扬为至高的艺术，一种永恒的美，矿区中回荡着凿石的低语，与对终结的向往。

        7. 终末百日“派对”

          这里聚集着一群沉沦末日谵妄的狂欢者，永无止境的派对在废墟中喧嚣，人们极端排外，却又狂热地向外宣传这堕落的救赎，笑声与尖叫交织成一曲扭曲的终曲，吸引着同样绝望的灵魂

        8. 梵蒂冈城国

          圣彼得的永恒堡垒，天主教世界的最后火种在此顽强闪烁，教皇的权杖高举，梦想重铸昔日信仰的荣光，在乱世中守护着那份神圣的延续。

        9. 圣马力诺共和国

          圣马力诺仍如战前般渺小而坚韧，这座山中古国在半岛的动荡中勉强维系独立，危机如阴影环伺，却以古老的城墙与智慧，悄然抵御着时代的风暴。

      2. 法兰西与低地

        第三次世界大战的硝烟尚未完全散尽，联合防御条约组织体系已然彻底解体，昔日盟誓如同秋风落叶悄然凋零。

        1. 法兰西共和国

          年逾七十二的马克龙依然牢牢握着总统权柄，战时铸就的中央集权如钢铁般日益坚固。然而，内部腐败如暗潮汹涌，贪婪与权斗侵蚀着国家肌体，民生凋零，民众在饥饿与绝望中挣扎，昔日荣耀的光辉已黯淡成残影，共和国如一艘在风暴中摇晃的巨舰，命运悬于一线。

        2. 低地联合防御管制区

          法兰西军队长驱直入比利时全境，建立起严密的军事管制。表面秩序井然，实则民怨沸腾，独立的情感如地下暗流涌动，人们怀念昔日自主的时光，渴望挣脱外来的铁链，那份蓄积已久的火焰，只待时机便将熊熊燃起。

        3. 东部联合防御管制区

          卢森堡与法兰西达成合作，建立联合防御管制区，将阿尔萨斯洛林并入，并进一步蚕食莱茵兰部分德国领土，形成一道漫长的军事缓冲。管制区内，居民对法兰西的统治既畏惧又不满，独立的呼声日益高涨，而东方德意志残余势力的阴影如乌云压境，威胁随时可能化作雷霆，这片土地已成为风云交汇的危局前沿。

        4. 摩纳哥公国

          摩纳哥表面仍如战前般优雅从容，地中海的碧波轻抚着赌场与游艇的奢华外衣，法兰西的条约庇护看似坚不可摧。然而战火余波已悄然渗入这弹丸之地，物资短缺如暗影笼罩，富豪阶层虽仍追逐浮华，底层居民却在高墙之内忍受着配给与恐慌，昔日不夜城的灯火如今黯淡许多，危机四伏的欧洲大陆让这座明珠也难以真正安然，随时可能被卷入更大的风暴。

        5. 安道尔公国

          (由于与法兰西共和国的紧密联系，这里将安道尔公国与法兰西与低地地区一并讲了。)

          隐于比利牛斯山脉的山国，由法兰西悄然代管政务，行政与外交皆奉承法国意志。谷地中的小镇依旧平静，居民的生活节奏如山泉般缓慢，这里远离战乱的喧嚣，成为乱世中一处难得的静谧角落。

        6. 尼德兰王国

          尼德兰人曾经与法兰西同结防御盟约，共抗外敌，却在战局变动中毅然退出联盟。然而退出代价惨重，南部大片富饶土地已被比利时在法兰西授意下占据，王国疆域从此残缺，北部残存的国土虽努力维持独立与尊严，却始终难忘那份被背叛的痛楚与失地之殇。

      3. 不列颠尼亚

        孤悬海外的帝国散尽了最后一丝余晖，垂暮的太阳终将落下……吗？不，大英帝国的人民绝不会容许这样的结局。在联合防御条约组织瓦解后，行将就木的军政委员会仍然顽固地把持着帝国的权柄。可内部的矛盾却愈演愈烈。庙堂之上，科技派与帝国派的斗争已经趋向白热化，老工业区的工人呼唤平等与自由，曾遭受饥荒蹂躏的国度对故土伸出双手。自那个至暗时刻以来，不列颠尼亚又一次走到了深渊边缘。

        1. 大不列颠王国

          名义上承袭古老王冠的合法继承者，实权却紧握于军政委员会之手。脱离大陆体系后，资源危机如铁链缠身，黑市中违禁品肆意流通，军政疏于管束，地方民间武装如野火蔓延，分离主义的暗流汹涌激荡。委员会内部，传统派固守离岸平衡的旧则，誓以稳定守护岛屿核心；科技派则追逐机械与创新的幻光，梦想借科技之力重铸霸权，化解内忧。派系倾轧与地方叛火交织，王国如一艘在雾海中分裂的巨舰，前路渺茫。

          (编者的话：科技派与萨默塞特地区某兄弟会暗中进行的实验将挑起不列颠内战的战火。)

        2. 爱尔兰国民国

          翡翠岛上，共和国经历剧烈改组，右翼力量强势崛起，新政府以铁腕重塑国家意志，民族主义的火焰在绿野间熊熊燃烧，悄然铸就一股不容忽视的岛屿新势。

        3. 国民北爱尔兰自治区

          北爱尔兰落入爱尔兰掌控之下，成为一处名义自治的边陲。尽管都柏林曾放弃对北爱的历史宣称，如今却以胜利者姿态重掌这片土地；伦敦的王冠仍高声宣称主权，争议的阴云如永不消散的雨雾，笼罩在这片分裂的土地之上。

      4. 伊比利亚

        曾一度登上日不落宝座的帝国业已没落，这个半岛如今身陷囹圄。在南方，阿拔斯王朝的遗孤卷土重来，天主的羔羊却求不来硫磺与天火的绝罚。塞巴斯蒂安主义者占领了里斯本，力图恢复大航海时期的荣光。这里曾是领航世界的先锋，五百年前是他们开拓了新航线。这里闪耀国际纵队的荣光，一百年前是他们反抗恶魔的蚕食。而现在，伊比利亚再一次走到了十字路口，我们不禁问道：丧钟为谁而鸣？

        1. 西班牙王国

          (此处指原西班牙王国政权。)

          西班牙政府衰微如残烛，兵力羸弱难振，外国干涉的触手在暗中纠缠，宫廷与边疆皆陷无力，王国如一具空壳，在风雨中勉强支撑。

        2. 安达卢西亚伊斯兰阴影领

          南方炽热大地升起新月旗帜，受阿拉伯伊斯兰联盟扶植的摩尔后裔在此重建格拉纳达的荣光，伊斯兰的呼声如沙漠风暴席卷，野心直指重征服整个伊比利亚，阴影中酝酿着昔日帝国的幽灵。

          (编者的话：在三战中阿拉伯人联合了起来，在后续我们将发布有关阿拉伯伊斯兰联盟的内容。)

        3. 永恒烈焰复兴骑士团

          西班牙王国在乱世中铸就的钢铁支柱，这支骑士团并非承袭古老的中世纪遗风，而是以现代化组织铸就的当代武装力量，纪律严明、结构高效，如一台精密运转的战争机器。他们高扬“永恒烈焰”的旗帜，象征信仰与意志的不灭燃烧，与南方安达卢西亚伊斯兰阴影领的对峙如两股烈火在焦土上碰撞，誓以不屈的永恒之焰守护王国残存的十字荣光。

        4. 西班牙王国

          (此处指位于巴斯克地区之卡洛斯派西班牙王国政权。)

          西班牙王冠的另一位继承者，拥戴卡洛斯血脉的政权，以更严苛的封建专制铸就铁律，宫廷阴谋与贵族威严交织，试图以绝对王权重塑半岛的秩序，却在乱世中孤立而傲慢。

        5. 苏维汇王国圣团联邦

          北方山脉与古城中，分离主义的火焰点燃苏维汇王国的旧梦，这片联邦以兄弟会般的誓约维系，复兴古老王国的荣光，成员间的情谊如钢铁般坚韧，在分裂的半岛上悄然筑起独立堡垒。

        6. 葡萄牙深渊民族复兴阵线

          塞巴斯蒂安主义的狂热信徒高举大航海时代的旗帜，誓以深渊般的意志复兴昔日帝国荣光，这支阵线如狂潮般席卷西部，已攻陷里斯本的金色港湾，梦想的幽影笼罩大西洋的波涛。

        7. 葡萄牙共和国

          共和政府的最后堡垒，如今里斯本沦陷，残存力量在危机四伏中苦苦支撑，民主的灯火摇曳欲灭，面对复兴阵线的铁蹄与民族主义的狂潮，前路渺茫而凶险。

      5. 希尔德海岸

        我们早已一无所有，并不害怕再次失去。在那近乎终结人类文明的战争中，德意志地区遭到了最沉重的打击，曾经繁华的都市沉入海底，只留一片废土，其上的人民苟延残喘。因而，这片土地上，古老的灰烬再度重燃。从神圣罗马帝国的末裔到马克思主义者，从乌托邦到警察之国。人类历史的缩影在这片土地上演，无数人想要毁灭德意志民族，但是我们终将重新崛起，这一次，我们不再依赖任何力量，我们仅有自己。

        1. 巴伐利亚自由王国

          阿尔卑斯脚下的狂想之国，人人皆可自冕为王，平等的王冠在啤酒泡沫与山风中闪耀，昔日秩序在此被彻底颠覆成一场永不落幕的嘉年华。

        2. 符腾堡议会王国

          议会殿堂中，君主由选票铸就，民主的仪式与王权的威严奇妙交织，选民的意志如钟摆般决定王座的归属，传统与民意在此达成脆弱的平衡。

        3. 霍亨索伦至高国

          霍亨索伦血脉高举旧日鹰旗，与资本巨头联手，梦想重塑神圣罗马帝国的辉煌，贵族的冠冕与金钱的权杖并肩，野心在莱茵河畔悄然复苏。

        4. 巴登大公国

          君主制在此焕发青春，宝座不再属于古老血统，而是属于青年代表的热血与理想，新一代的旗帜在黑森林中猎猎作响，象征着破晓前的曙光。

        5. 赖特亚“新罗马”
